### Libraries used:

- React with TypeScript
- React-Router
- Material-UI
- recharts
- Jest

## Install packages

```
npm i
```

## Run in development mode...

```
npm start
```

## ...or run builded...

```
npm run build
npm install -g serve
serve -s build
```
